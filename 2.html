02. TAG

  • '<'과 '>' 안에 들어가는 걸 'tag'라고 표현한다.
  • 태그 중에는 단독적으로 쓰이는 것과, 오픈 태그 + 클로즈 태그 형식으로 쓰이는 것이 있다.
  • 오픈 태그는 단독적으로 쓰이는 것처럼 쓰면 되고, 클로즈 태그는 '<' 뒤에 '/'를 입력하면 된다.


가장 기본이 되는 태그:

  • document type -> !doctype ____ (밑줄 안에는 html과 같은 파일 타입을 입력하면 된다)
  • html -> html
    • 클로즈 태그가 있다.
  • head -> head
    • 클로즈 태그가 있다.
  • body -> body
    • 클로즈 태그가 있다. 
    HTML 파일은 'head'와 'body'로 크게 나눌 수 있다.


HEAD

  • head에는 'title', 'meta charset'과 같은 태그를 사용한다.
    • 'charset'은 'character set'을 의미한다.
  • title의 오픈 태그와 클로즈 태그 사이에 기입하는 내용은 tab에 표시된다.
  • meta charset 태그는 내가 코딩할 때 파일에 사용한 인코딩 방식을 알려줌으로써, 내가 기입한 내용이 웹사이트에 잘 반영될 수 있도록 써준다.
  • 'utf-8' 인코딩 방식을 사용하는 경우, '< >' 안에 meta charset 태그 다음에 '=utf-8'라고 적으면 된다.


BODY

Body에서 자주 사용되는 태그:

  • heading -> h1, h2, ... , h6 (총 6개 존재)
    • 클로즈 태그가 있다.
  • paragraph -> p
    • 클로즈 태그가 있다.
  • anchor -> a (link 용도로 사용)
    • 클로즈 태그가 있다.
  • unordered list -> ul (bullet point 용도로 사용)
    • 클로즈 태그가 있다.
  • ordered list -> ol (numbered list 용도로 사용)
    • 클로즈 태그가 있다.
  • image -> img
    • img 태그 안에 src (source) 표시를 해줘야 한다.
    Ex. img scr="_________" width=100%
(밑줄에는 사진 파일 이름.확장명을 적는다.)
  • width -> width (사진의 크기를 픽셀수나 %로 정할 수 있다)
  • line breaker -> br
  • indentation -> indent:__px (밑줄 안에는 픽셀 숫자를 입력하면 된다)
  • bold -> b
    • 클로즈 태그가 있다. 'strong' 태그도 같은 효과.
  • emphasis -> em (이탤릭체)
    • 클로즈 태그가 있다.
  • underline -> u
    • 클로즈 태그가 있다.
  • checkbox -> checkbox
