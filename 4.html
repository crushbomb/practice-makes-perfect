04. Web Hosting

직접 만든 HTML 파일을 다른 컴퓨터에서 접속할 수 있으려면 웹 호스팅 해주는 곳을 이용하는 방법이 있다. GitHub을 이용하면 무료로 사용해볼 수 있다.
GitHub을 눌러서 로그인을 해보자.

  1. 'create a new repository'는 새로운 작업공간을 만드는 것과 비슷하다.
  2. 새로운 공간을 만들어서 이름을 지어주자. Ex) practice makes perfect
  3. code 탭 하단에 'creating a new file' or 'uploading an existing file'이 있다.
    'uploading'을 선택한 후 코딩해놓은 파일들을 드래그 하거나, 'choose your files'를 눌러서 코딩한 파일들이 위치한 폴더를 찾고, 파일들을 전부 선택해 업로드 해준다.
  4. 업로드가 다 되면, Settings 탭 -> 사이드바에 Pages로 가서 'Branch' 아래에 'none' -> 'main'으로 바꾸고 Save 해준다.
  5. Actions 탭으로 이동해보면 내가 업로드한 파일들을 웹사이트로 만드는 작업 현황을 실시간으로 체크할 수 있다.
  6. 완료가 되면, Settings -> Pages로 다시 가보면, 'Your site is live at' 뒤에 웹사이트 링크가 생겼다. 링크를 눌러서 잘 만들어졌는지 확인해보자.
  7. 나중에 HTML 파일 등을 수정했다면, code 탭에서 다시 모든 파일들을 다시 업로드해주면 된다.
  8. 업로드가 완료되면, 자동으로 웹사이트를 업데이트 해준다. 작업 현황을 Actions 탭에서 확인하고, 마무리가 되면 Settings -> Pages에 뜨는 링크로 들어가서 다시 확인해주자.
